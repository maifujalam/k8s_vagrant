1. Installation:-
    a. curl -L -O https://download.falco.org/packages/bin/x86_64/falco-0.36.2-x86_64.tar.gz
    b. tar -xzvf falco-0.36.2-x86_64.tar.gz
2. Falco will be already present in CKS env.
3. journalctl -fu falco [f: follow, u: unit]
4. Validate falco:-
    a. Notify in falco logs  if exec into container:
        i. k exec -it nginx -- bash
        ii. - rule: Detect shell inside container
              desc: Alert if shell inside container is opened.

    b. Notify if try to access shadow file/
    c. Notify if try to delete /etc/audit.log
    d. Notify if
5. Write falco rules( 5 mandatory keys):-
   - rule: <uniq name of rule>
     desc: <detailed description of rule>
     condition: <condition to trigger rule>
     output: <puput to generate for event>
     priority:
6. Check spanned shell:  cat /var/log/syslog | grep shell
7. Find falco rule:  cd /etc/falco then grep -ri "shell in"
8. grep -ri -B 3 -A 3 "shell in"
9. Grep reccursively in a folder with matching string and line number with starting and endoing 3 line:
   grep -ri -n -A 3 -B 3 "shell in"
10. crictl ps -id
11. crictl pods -id
12. time : evt.time
    time without nanoseconds: %evt.time.s
    user-name: %user.name
    user-id: %user.uid
    container-id: %container.id
    Container name: container.name
    container-image: container.image
    container-image-repository: container.image.repository
    kubernetes namespace: k8s.ns.name
    kubernetes pod name: k8s.pod.name
 13. Copy falco: cut -d " " -f 9 /opt/falco/raw-falco /opt/falco/falco-output.txt
